# &#x2754; Что реализвано в проекте?

Основные функции:
    - Сохранение фалов, при нажатии сочетании клавиш `CTRL + S`
    - Подсветка синтаксиса.
    - Имеет терминал, но до интерактивного ему далеко.

Может быть, появиться в будущих версиях:
    - Вкладки, с файлами.
    - Интерактивный терминал.
    - Возможность добовлять новый функционал редактору, с помощью расширений.
    - Больше тем.
    - Улучшенный UI.

# &#x2757; Что важно учитывать при использовании данного проекта.
Надо обязательно учитывать:
    &#x26AB; Проект создан, для улучшения работы с UI библиотеками(именно автору, для него - ценный опыт).
    &#x26AB; Проект чисто для учёбы(да это сейчас, но в дальнейшем, попытаемся воссоздать функционал, котоырый, возможно, сможет конкурировать с мощными IDE.)
